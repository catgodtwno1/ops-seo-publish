# ops-seo-publish

多平台 SEO 文章發布自動化技能，適用於 OpenClaw 瀏覽器自動化場景。

## 功能

- **四大平台自動發文** — 百家號、知乎、搜狐號、簡書
- **編輯器適配** — 針對每個平台的富文本編輯器（UEditor / Draft.js / Quill）做了專門處理
- **防封策略** — 發文間隔、User-Agent 偽裝、行為模擬
- **封面圖處理** — 自動上傳/設定封面圖（百家號必填）
- **Chrome 多 Profile 隔離** — 每個平台獨立 Chrome Profile，避免 cookie 衝突

## 適用場景

- 批量發布 SEO 文章到多個中國內容平台
- 企業品牌內容分發
- 需要自動化處理不同編輯器的文章發布

## 平台每日限額

| 平台 | 每日限額 | 百度 SEO 權重 |
|------|---------|--------------|
| 百家號 | 5 篇（無原創認證）| ⭐⭐⭐⭐⭐ 最高 |
| 知乎 | ~5 篇（新號）| ⭐⭐⭐ |
| 搜狐號 | 5-10 篇 | ⭐⭐⭐⭐ |
| 簡書 | 2-5 篇 | ⭐⭐ |

## 前置條件

1. Chrome 已安裝並啟用遠端調試
2. 各平台帳號已登入對應的 Chrome Profile
3. OpenClaw 已配置 `browser.profiles`（多 Profile 端口映射）

## 安裝

將此目錄放到 `~/.openclaw/workspace/skills/` 下，OpenClaw 會自動載入。
